Taruh file label YOLO (.txt) di folder ini.
Nama file harus sama dengan gambar di folder ../images/.

Format setiap baris: <class_id> <cx> <cy> <width> <height>

Kelas:
  0 = person
  1 = phone
  2 = laptop
  3 = book
  4 = suspicious

Contoh isi label:
  0 0.512 0.421 0.280 0.650
  1 0.750 0.820 0.080 0.120
